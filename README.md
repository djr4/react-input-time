# input-time

## Demo
https://djr4.github.io/react-input-time/

## Install

```bash
npm install --save input-time
```

## Usage

```jsx
import InputTime from 'input-time'

<InputTime
  onChange={(e) => { console.log(e)}}
  value={'10:00'}
  label={'Opening hours'}
  inputStyle={{ width: 20, }}
/>
```
